# Spacing & Layout

## Spacing Scale

```
4px   (space-1)
8px   (space-2)
12px  (space-3)
16px  (space-4)
20px  (space-5)
24px  (space-6)
32px  (space-8)
40px  (space-10)
48px  (space-12)
64px  (space-16)
```

These map directly to Tailwind's default spacing utilities (`p-1` through `p-16`).

## Layout Rules

- generous spacing between sections
- clear grouping
- avoid clutter
- precise alignment
- readable content width

## Borders

- subtle, thin
- primary separation mechanism
- use `border-border-default` for standard borders

## Border Radius

| Token | Value | Use |
|---|---|---|
| `rounded-sm` | 6px | inputs, buttons |
| `rounded-md` | 8px | inputs, buttons, small cards |
| `rounded-lg` | 12px | cards, modals |

## Shadows

- minimal usage
- often unnecessary
- prefer borders over shadow for elevation
